# Example Output: Pedal Build Pack

This directory demonstrates the artifact tree generated by v2.0 of the Pedal BOM Manager.

## Contents
* **Field Manuals/**: Compiled PDF build guides.
* **Sticker Sheets/**: 1:1 scale drill templates and enclosure art.
* **Source Documents/**: Original raw BOMs used for this generation.
* **Pedal Shopping List.csv**: Aggregated procurement list.
